# Kost

Kost is a self-hosted application for shared household expenses.
It includes a NestJS API, a React web app, and PostgreSQL.

## Quick Start

```bash
git clone https://github.com/OnlyDrey/Kost.git
cd Kost
cp .env.example .env
npm install
docker compose up --build
```

Then open:
- Web UI: http://localhost:3000
- API docs: http://localhost:3000/api/docs
